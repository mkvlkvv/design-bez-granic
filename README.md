# Путешествия - лучший способ быть вместе 

## Задание:
Поиск автомобильного маршрута

Сайт: [путешествуем.рф](https://путешествуем.рф)

Пользовательский сценарий - Найти автомобильный маршрут
   - использовать фильтры поиска
   - открыть карточку маршрута
   - получить информацию о поездке

## Решение
- Исходный код нашего сайта - [Папка layout-for-screenreader](https://github.com/mkvlkvv/hackaton/tree/page-screenviewer)
- Развёрнутый статический сайт с примером страницы фильтров и одниз из маршрутов - [hackaton-6f6eb.web.app](https://hackaton-6f6eb.web.app/)
- Дизайн фигмы адаптированный под простой русский язык и контрастность - [figma.com](https://www.figma.com/design/tvfdToYEJFslu3senoa8PI/%D0%9F%D1%83%D1%82%D0%B5%D1%88%D0%B5%D1%81%D1%82%D0%B2%D1%83%D0%B5%D0%BC.%D1%80%D1%84?node-id=0-1&t=yw1OgK6pt5gLJAeg-1)
